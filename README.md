# E2E Test Results

This branch contains e2e test results published by GitHub Actions.

## Accessing Results

Results are organized by run ID under the `runs/` directory.

Raw content URLs:
```
https://raw.githubusercontent.com/bjuergens/claudetest123/test-results/runs/<run_id>/test-results/summary.json
https://raw.githubusercontent.com/bjuergens/claudetest123/test-results/runs/<run_id>/test-results/screenshots/<name>.png
https://raw.githubusercontent.com/bjuergens/claudetest123/test-results/runs/<run_id>/test-results/logs/<name>.log
```
